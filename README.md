# Mini Quora - RESTful CRUD Application

## Description
A full-stack web application built using Node.js, Express, and EJS.
Users can create, edit, view, and delete posts following REST principles.

## Features
- Create new posts
- Edit existing posts
- Delete posts
- View post details
- RESTful routing implementation

## Tech Stack
- Node.js
- Express
- EJS
- UUID
- Method-Override

## How to Run
1. Clone the repository
2. Run npm install
3. Run node app.js
4. Open http://localhost:3000/posts
